# Web Automation Orchestrator

Use `web-automation-orchestrator.md` as the canonical project instruction for web automation tasks.

## Antigravity Adapter Rules

- Paste the full contents of `web-automation-orchestrator.md` into Antigravity's project or custom instruction area if file imports are unavailable.
- If Antigravity supports project instruction files, store this adapter beside `web-automation-orchestrator.md` and configure the project to load both.
- Preserve the orchestrator workflow: analyze task complexity, validate tools, execute with the selected automation tool, and report results in the required output format.
- Prefer DOM, accessibility tree, and semantic snapshots over screenshot-first workflows.
- Use Antigravity's native tools and permission model.
- Communicate in Indonesian unless the user asks otherwise.
